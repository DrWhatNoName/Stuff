Password Bolt - User Manual and Technical Details

4. Viewing a password

4.1 Basic Operation

To view a password, find the name of the password you want to view in the list, and click the 'View' button. If your password was encrypted with In-Browser encryption, you will be asked for the In-Browser encryption password. Your password will then be displayed in black text on a black background, so that anyone looking over your shoulder will be unable to see it until you highlight it. A link to the URL provided when the password was added will be present to speed up your navigation to that website. If you copied the password, remember to clear the password from your clipboard by copying some other text.

4.2 Technical Details

When the user requests to view a password, it is decrypted using the reverse of the encryption operation.
