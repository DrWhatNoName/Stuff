Password Bolt - User Manual and Technical Details

10. Changing the account password

10.1 Basic Operation

To change the password for your account, click 'Settings' on the navbar. In the 'Old' section, enter your old password and keyfile. In the 'New' section, enter your new password and keyfile and click 'Change Password'. You will now be able to view your passwords by logging in with the new password and keyfile. If there were already passwords saved under the new password, they will still be there.

10.2 Technical Details

When the user changes his password, the passwords and notepad in the database are decrypted and encrypted again with the new password. The authentication token is also updated for the new password.
