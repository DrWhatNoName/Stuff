Password Bolt - User Manual and Technical Details

8. Using the notepad

8.1 Basic Operation

Password Bolt includes a notepad where you can save any kind of data. To get to the notepad, click 'My Notepad' in the navbar. Enter the text you want to save in the notepad into the textbox and click 'Save'.

8.2 Technical Details

The notepad is stored in the same way as the passwords. The MySQL's SHA1 hash function is used to provide plausible deniability. The notepad is encrypted in the same way as the passwords.
