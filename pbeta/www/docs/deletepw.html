Password Bolt - User Manual and Technical Details

6. Deleting a password

6.1 Basic Operation

To delete a password, find it in the list then click 'Delete'. Click 'Delete' again to confirm that you want to remove the password.

6.2 Technical Details

When the user deletes a password, it is simply removed from the database.
