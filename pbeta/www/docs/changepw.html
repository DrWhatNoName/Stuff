Password Bolt - User Manual and Technical Details

5. Changing a password

5.1 Basic Operation

To change a password, find it in the list and click the 'Change' button. In the 'Change Password' section, the data for your password will already be in the textboxes. Change the information you want, then click the 'Change' button at the bottom of the section.

If your password is encrypted with In-Browser encryption, the password will not be in the password box, you will need to view it first, copy it, and re-enter the In-Browser encryption password before clicking 'Change'.

5.2 Technical Details

When the password is changed, all of the information is encrypted again using the same methods as adding a new password.
