Password Bolt - User Manual and Technical Details

9. Logging out

9.1 Basic Operation

It is very important to log out from password bolt when you are done using it. Click 'Logout' in the navbar.

9.2 Technical Details

When the user logs out, the cookies storing the encryption key are expired, and the cookie encryption key is removed from the database. The user is automatically logged out if his IP address is not the same as the one he logged in with, and after the session time expires.
